## Installing Latest Version
```lua
local VexUI = loadstring(game:HttpGet("https://github.com/SSHRKs/VexUI/releases/latest/download/VexUI.lua"))()
```
## Installing Selected Version
```lua
local Version = "1.0.0"
local VexUI = loadstring(game:HttpGet("https://github.com/SSHRKs/VexUI/releases/download/".. Version .."/VexUI.lua"))()
```
